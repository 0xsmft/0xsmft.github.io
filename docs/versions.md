Starting in Saturn version 0.2.6, each version now comes with it's own build unqiue ID (UBID).

### UBID

A UBID is randomly generated upon a final build before distributing the Engine. 

A UBID can be 5 characters long or 6 characters long.
The format must look like X99(9)XX e.g. A26AA or A260AA
